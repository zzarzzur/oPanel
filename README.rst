This Is A Web Panel Like cPanel But Open Source
        It Is Called oPanel

Installation
        Run this command::
                sudo mkdir /usr/local/oPanel/ && sudo cp -rf * /usr/local/oPanel/ && sudo ln -s /usr/local/oPanel /var/www/oPanel;
        Then edit this file::
                /usr/local/oPanel/config.phpdb
        Then Navigate To::
                http://localhost/oPanel/
